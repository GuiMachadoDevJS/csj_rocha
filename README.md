# CSJ Rocha 

## Estrutura de Arquivos

```
csj-rocha /
├── index.html          ← Página principal
├── css/
│   └── style.css       ← Identidade visual azul marinho + branco
├── js/
│   └── main.js         ← Preloader, navbar, parallax, form, animações
├── img/                ← ⚠️ Adicione as imagens aqui
│   ├── logo.png            ← Logo CSJ Rocha (PDF 2, pág. 1)
│   ├── hero-bg.jpg         ← Foto dos caminhões em operação (fundo do hero)
│   ├── sobre-operacao.jpg  ← Foto para seção Sobre Nós
│   ├── munck-mercedes.jpg  ← Munck Mercedes-Benz L-1620 (PDF 1, pág. 4)
│   └── munck-vw.jpg        ← Munck VW Constellation 24.250 (PDF 1, pág. 5)
└── README.md
```

## Caracteristicas

- **Paleta azul marinho** fiel ao portfólio da CSJ Rocha (#0B1F3A + #1565C0)
- **Preloader** animado com barra de progresso
- **Hero com grade técnica** e efeito parallax no scroll
- **Card flutuante** de frota no hero (desktop)
- **Ribbon strip** com informações rápidas abaixo do hero
- **Seções light/dark alternadas** para ritmo visual profissional
- Formulário → WhatsApp com toast de confirmação

## Como adicionar as imagens

1. Abra os PDFs e exporte as imagens (botão direito no Adobe Reader ou ilovepdf.com)
2. Renomeie conforme a tabela acima
3. Coloque na pasta `img/`
   
> O site funciona mesmo sem as imagens — placeholders são exibidos.
